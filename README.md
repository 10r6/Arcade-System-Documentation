# Arcade System Documentation

## Introduction
Playable Arcade System lets you place working arcade machines inside a 3D scene. Players can approach an arcade cabinet, interact with it, and play a mini-game rendered on the arcade screen.

The system uses a separate camera and a RenderTexture to display the game on the arcade monitor. It also includes an input switching system so the player controls are temporarily redirected to the arcade game while playing.

This allows developers to create interactive arcades, mini-game hubs, or playable props inside their worlds.

---

## Features
- Playable arcade machines inside a 3D scene
- RenderTexture based arcade screens
- Separate camera for each mini-game
- Input switching between player and arcade game
- Multiple arcade machines supported in the same scene
- Easy setup for creating new arcade games

---

## Requirements
- Unity 2022 or newer
- Universal Render Pipeline (URP)
- Unity Input System

---

## Typical Use Cases
This system can be used for:

- Retro arcade environments
- Game hubs with playable mini-games
- Interactive props inside a game world
- Virtual arcade rooms
- Multiplayer social spaces
